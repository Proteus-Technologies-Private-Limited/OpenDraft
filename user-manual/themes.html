Themes

Switch between light and dark interface themes.

Switching Themes

Go to View > Light Theme or View > Dark Theme to switch between the two interface themes.

Light Theme

The default theme. The interface uses a bright, clean design with light grays and whites. The editor content area has a white page background.

Dark Theme

A dark interface for writing in low-light environments or for reduced eye strain during long sessions. The toolbars, panels, menus, and dialogs use dark backgrounds with light text.

ℹ️

Note: The editor page itself stays white in both themes, matching a real printed page. The dark theme applies to the surrounding interface (toolbar, sidebars, dialogs, menus).

Theme Preference

Your theme preference is saved automatically and persists between sessions. The next time you open OpenDraft, it will use the last theme you selected.
