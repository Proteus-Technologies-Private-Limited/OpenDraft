Keyboard Shortcuts

Quick reference for all keyboard shortcuts in OpenDraft.

ℹ️

Windows/Linux users: Replace ⌘ (Cmd) with Ctrl for all shortcuts.

File

ActionShortcut
New Screenplay⌘N
Save⌘S
Export as PDF / Print⌘P

Edit

ActionShortcut
Undo⌘Z
Redo⇧⌘Z
Select All⌘A
Find & Replace⌘F
Go to Page⌘G

Formatting

ActionShortcut
Bold⌘B
Italic⌘I
Underline⌘U

Screenplay Elements

ElementShortcut
Scene Heading⌘1
Action⌘2
Character⌘3
Dialogue⌘4
Parenthetical⌘5
Transition⌘6
General⌘7
Shot⌘8

Tips

  • The element shortcuts (⌘1–8) are the most important to learn. They let you change element types without leaving the keyboard.
  • ⌘Z (Undo) is your safety net — use it freely.
  • ⌘F (Find) is useful not just for searching, but for navigating — type a character name to jump to their scenes.
  • You can also access keyboard shortcuts from Help > Keyboard Shortcuts inside the app.
