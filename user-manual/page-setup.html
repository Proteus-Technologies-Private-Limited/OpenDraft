Page Setup

Configure page dimensions, margins, and layout.

Opening Page Setup

Go to File > Page Setup to open the Page Setup dialog.

Settings

SettingDefaultDescription
Page Width 8.5 inches Width of the page (US Letter)
Page Height 11 inches Height of the page (US Letter)
Top Margin Standard Space above the first line of text
Bottom Margin Standard Space below the last line of text
Left Margin 1.5 inches Standard left margin for screenplays (allows for hole punches)
Right Margin 1 inch Standard right margin
Header Margin Standard Space for page headers
Footer Margin Standard Space for page footers
⚠️

Changing page dimensions affects your page count and timing estimate (1 page ≈ 1 minute). The standard US Letter (8.5×11") with default margins is the industry standard for screenplays.

Resetting to Defaults

Click Reset to Default in the Page Setup dialog to restore standard screenplay page dimensions and margins.
